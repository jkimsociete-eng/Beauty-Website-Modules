Avant de commencer

Le principe universel

Chaque module est un fichier HTML autonome. Pour l'intégrer, il suffit de copier son contenu dans un bloc "HTML" de votre éditeur.

✦
Ce que vous copiez : Ouvrez le fichier du module (ex: 08-quiz.html), sélectionnez tout (Ctrl+A), copiez (Ctrl+C), collez dans le bloc HTML de votre CMS.
✓
Vos images et vidéos sont hébergées sur Cloudinary — elles se chargent automatiquement. Vous n'avez rien à uploader sur votre site.
!
Ne pas utiliser le fichier 00-pack-complet.html pour l'intégration — c'est uniquement la démo navigable. Utilisez les fichiers individuels 01 à 13.

Adapter à votre marque

Dark & Light

Changer le thème

Chaque module supporte les 2 thèmes. Le bouton 🌙/☀️ switche tout en temps réel.

Forcer un thème fixe (sans toggle)
<!-- Thème sombre (défaut) --> <html lang="fr" data-theme="dark"> <!-- Thème clair --> <html lang="fr" data-theme="light">
✓
Le choix de thème est mémorisé automatiquement via localStorage — vos visiteurs retrouvent leur préférence à chaque visite.
Questions fréquentes

Problèmes courants

Le module ne s'affiche pas correctement
Vérifiez que vous avez copié tout le fichier (Ctrl+A puis Ctrl+C). Certains CMS tronquent le code si vous copiez partiellement. Sur Wix, agrandissez le bloc HTML.
Les vidéos ne se chargent pas
Vérifiez que vos URLs Cloudinary dans VIDEO_SRCS sont correctes et que vos vidéos sont bien publiques sur Cloudinary (pas en mode privé).
Les fonts ne se chargent pas
Les polices chargent depuis Google Fonts. Si votre site bloque les ressources externes, copiez le lien Google Fonts dans le <head> de votre thème plutôt que dans le bloc HTML.
Le module est trop petit sur mobile
Sur Wix et certains CMS, le bloc HTML a une hauteur fixe. Faites glisser le coin inférieur pour l'agrandir, ou ajoutez min-height: 600px au bloc conteneur.
Puis-je utiliser plusieurs modules sur la même page ?
Oui, mais attention : si vous utilisez 2 modules avec le même ID (ex: 2 quiz), renommez les IDs dans le HTML. Ex: id="qs1" → id="qs1-b" dans le second module.
